Implementace

Použité technologie:

Laravel, PHP, MySQL, Blade Templates, Bootstrap, JavaScript, jQuery

Routy:

/routes/web.php
Definice všech rout aplikace, které určují, jaké URL odpovídají jakým kontrolerům a akcím.

Controllery:

/app/Http/Controllers/
Obsahuje všechny controllery, které zpracovávají požadavky uživatelů a volají příslušné modely a pohledy.

  1. Controller - enumerace typu činnosti, odrůdy a stavu nákupu
  2. KosikController - správa košíku a vytvoření objednávek
  3. NakupyController - správa a detail objednávek
  4. OsetreniController - evidence vyřízení činností (ošetreni, řezy, postřiky, ...), detail ošetření, odstranění ošetření
  5. PlanovaniController.php - vyváření plánu pro ošetření a sklizně
  6. SklizenController - správa sklizní
  7. SpravaRadkuController - zobrazení seznamu a detailu řádku, vytvoření a úprava
  8. UserController - autorizace uživatelů
  9. UserManagementController - správa uživatelů
  10. VinoController - správa vín

Modely:

/app/Models/

Obsahuje všechny modely reprezentující tabulky v databázi a jejich vztahy.

Migrace:

/database/migrations/
Obsahuje všechny migrace, které definují strukturu databázových tabulek.

Seedery:

/database/seeders/
Obsahuje seedery pro naplnění databáze počátečními daty.

Pohledy:

/resources/views/
Obsahuje všechny Blade šablony pro generování HTML stránek.

Middleware:

/app/Http/Middleware/
Obsahuje middleware, které zpracovávají HTTP požadavky před tím, než jsou předány do kontrolerů.

Výčtové typy:

/app/Enums/
Obsahuje výčtové typy.

Databáze

Instalace

  1. > npm install
  2. > composer install
  3. Přejmenovat .env.example na .env
  4. Ve vzniklém .env souboru nastavit připojení na databázi (DB_*)
  5. > PHP artisan key:generate
  6. > PHP artisan migrate
  7. > PHP artisan db:seed
  8. > PHP artisan serve
